헬스케어 서비스를 유지보수하며 BLE 장비 연동 최적화 및 트러블슈팅 경험이 있습니다. 또한 디지털치료기기(DTx) 프로젝트를 다수 진행하며 알코올중독, 경도인지장애, 약물중독(마약) 환자를 대상으로 한 Android Application 개발을 진행, 배포 경험이 있습니다.

Android 영역에만 머무르지 않고, Ktor 기반 REST API 서버를 직접 구축·운영해 본 경험을 바탕으로 클라이언트와 서버를 아우르는 전체 시스템 관점에서의 소통과 설계가 가능합니다.

현재는 단순한 앱 개발을 넘어, 고성능 모바일 환경에서 안드로이드 클라이언트가 수행할 수 있는 기능의 한계를 넓히는 데 몰입하고 있습니다. 특히 **NDK(C++) 를 사용한 데이터 전처리, 온디바이스 모델 추론** 기술에 주력하고 있으며, 모바일기기 내에서 이미지나 오디오 등 다양한 데이터를 직접 가공하고 딥러닝 모델로 추론하여 결과를 사용자에게 제공하는 개인 프로젝트를 통해 관련 역량을 키우고 있습니다.
